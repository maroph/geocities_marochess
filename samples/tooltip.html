Samples: A Simple CSS Based Tooltip System

In the following text I use a very simple CSS based tooltip system: A tooltip is related to this textThis is the related tooltip. There might be better solutions but this one is very simple to use.

CSS code for this solution:

.tooltip {
    position:relative;
    z-index:24;
    background-color:#d0d0d0;
    color:#000000;
    text-decoration:none;
}

a.tooltip:hover {
    z-index:25; 
    color:#000000;
    background-color:#d0d0d0;
}

a.tooltip span {
    display: none;
}

a.tooltip:hover span {
    display:block;
    position:absolute;
    padding:5px;
    top:2em;
    left:2em; 
    width:15em;
    border:1px solid #000000;
    background-color:#f0f0f0; 
    color:#000000;
    text-align:center;
    font-family:Verdana, Helvetica, sans-serif;
    font-size:80%;
    font-style:italic;
}

HTML code for this solution:

<a class="tooltip" href="#">A tooltip is related to this text<span>This is the related tooltip</span></a>.
