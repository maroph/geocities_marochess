Assignment 3: Animation and jQuery

On this page you will find the links to my assignment 3 data.

  • assignment3-ManfredRosenboom
    Online version of my assignment.
    • You will see 8 squares which represent in total 4 drum and 4 bass sounds.
    • The layout is hard coded to show 2 lines with each 4 squares.
    • You can play a sound by clicking on the square or by pressing the shown key on your keyboard.
    • If the audio object isn't available in your browser you will see a warning and the sound squares aren't drawn.
    • Underneath the squares you will see an animation area (labeled "Simple Global Animation") which will show the color of the chosen sound to play.
    • Under this are is an area (labeled "Simple Per Sound Animation") which contains an animation per sound. The related code is taken from the demo shown in the assignment description.
    • At the bottom of the page you will see a history line. The size of history line is limited by the variable maxHistSize (at the top of the file main.js) to a value of 10.
  • assignment3-ManfredRosenboom.zip
    ZIP file of the above hyper linked directory. The includes node.js web server is listening on port 2014.

I have tested my assignment data on my Windows 7 32bit PC with the following browsers:

  • Chrome 34.0.1847.116
  • Firefox 28.0
  • Internet Explorer 11.0.9600.17041
