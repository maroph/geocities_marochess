Assignment 2: HTML5 Canvas Paint Application

Subject 2: for people with JavaScript knowledge (ones with some experience in developing webapps with more than 40 lines of JS!)

Start from the source of the paint program provided in the wrap up section at the end of this chapter. Add some features to this application for drawing text, drawing other shapes (triangle), draw with texture or with a gradient fill or stroke style.

Your application must be online somewhere and you will provide a Web page with some explanations of your additions + a downloadable archive of your version of the paint program.

Description

My code is based on step 6 of the tutorial used in the course material.

Added Drawing Tools

  • ellipse
    Draw an ellipse with the center set by the first mouse down click.
  • gradRectangle
    Draw a rectangle filled by a rainbow gradient.
  • image
    Add an image (loaded by URL) to the canvas.
    By clicking on this draw tool, the desired URL is prompted. A default image is preselected and can be used. With the next mouse down the image will be visible and draggable on the canvas until mouse up.
  • text
    Add a text to the canvas.
    By clicking on this draw tool, the desired text is prompted. A default text is preselected and can be used. With the next mouse down the text will be visible and draggable on the canvas until mouse up.
  • triangle
    Draw a triangle: first mouse down sets 1st corner
    1st mouse up sets the 2nd corner and the
    2nd mouse up closes the triangle.

Other Added Features

  • Highlight (mark) currently selected drawing tool.
    The currently selected drawing tool is highlighted.
  • Add tool tips to the drawing tool names.
  • Button: "Clear Canvas".
    A click on this button simply clears the canvas.
  • Download link
    Makes the current canvas state available for download as an image.
