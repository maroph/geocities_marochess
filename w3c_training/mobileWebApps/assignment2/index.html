Assignment 2: Mobilize the Diary Application

I viewed the web app with the following devices:

  • Samsung Galaxy S4
  • Kindle Fire HD
  • Apple iPad air
  • Windows 7 PC 24inch monitor with the current Chrome and Firefox versions

Part 1: Play around with given assignment-3.1.html file

  • assignment-3.1.html (JavaScript)
    This JavaScript solution adds an click event listener to each title. Clicking on an title line will toggle (show/hide) the related text.
  • assignment-3.1_2.html (CSS hover)
    This solution shows the text when the mouse is hovered over a title line.
    This hover solution is very compact but of course of no use on a mobile device (no mouse hovering).

Part 2: Mobilize the Diary Application

This is my mobilized version of the given MyDiary web application. The web application is now mobile device aware.

The page uses a min width of 250px. This value should be okay for today's mobile phones.

Missing feature: you can't (same as in the original web application) delete single entries of the diary.

You can use the web application via 2 URLs:

  • diary/index.html
    (Mobilized Diary)
  • diary/index.html?clear
    (Mobilized Diary with data removal if entries are available)
