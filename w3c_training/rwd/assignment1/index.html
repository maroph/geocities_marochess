Changes

  • Adapt to HTML5 markup
    I have adapted the pages index.html, rooms.html, prices.html, and contact.html to HTML5. For the rest of the pages I adapted the HTML header only.
  • All tables are rewritten to CSS based code.
  • All CSS related code is located in the file style.css
  • All JavaScript related code is in the file bnb.js
  • JavaScript disabled
    • The promo page will be viewed at the top of the home page in an iframe. As an alternative (commented in the index.html file) it will only shown as a link at the top of the page.
    • All "Close" buttons won't be displayed.

Open Questions/Problems

  • Promo Poupup
    With JavaScript enabled I use simply the following code at the bottom of the index.html file:
    window.addEventListener('load', showAdvertisement, false);
    Maybe that isn't what was the intention of this assignment. But it isn't clear to me, what instead is expected to do when loading the home page of the sample site.
  • IE8 Support
    I have added html5shiv.min.js, as described, to the the pages index.html, rooms.html, prices.html, and contact.html. As I have only IE11 on all my machines, I can't do any IE8 related tests.
    To be honest: As I use a lot of modern techniques I'm not that much interested in supporting that old browser versions.
